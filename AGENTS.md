## Project Summary
A comprehensive Tour Management System (TMS) designed to manage tours, announcements, attendance, budgets, and safety protocols.

## Tech Stack
- **Frontend**: React.js (Next.js Framework)
- **Backend**: Node.js / Express.js
- **Database**: MySQL

## Architecture
- **Client**: Next.js (App Router) acting as the React frontend.
- **Server**: Express.js server for API logic and MySQL database interactions.
- **Project Structure**:
  - `/src`: Frontend React components and pages (Next.js).
  - `/server`: Express.js backend.

## User Preferences
- Tech stack MUST be React.js, Node.js/Express.js, and MySQL.

## Project Guidelines
- Follow clean code practices.
- Ensure type safety with TypeScript.

## Common Patterns
- Using Express for API endpoints.
- Using MySQL for persistent data storage.
